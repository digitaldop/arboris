Import estratto conto

Vai ai movimenti Dashboard

Questa è la modalità principale per importare i movimenti bancari: scarica l’estratto dalla tua banca (CAMT.053 XML o, se disponibile, CSV) e caricalo qui.

Arboris prova a riconoscere automaticamente formato, colonne, date, importi e conto bancario. Prima dell’import definitivo viene sempre mostrata un’anteprima.

Per evitare timeout sui file con molti movimenti, l’import registra prima i movimenti bancari. La riconciliazione delle rette si avvia poi dalla pagina dedicata.
